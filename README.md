# 贪吃蛇（Java 版）

一个使用 Java Swing 实现的可玩版贪吃蛇小游戏，支持手动操作和自动寻路两种模式。

---

## ✅ 快速开始（推荐）

1. 双击运行：`启动游戏.bat`  （假如你已经编译过一次）
2. 或双击运行：`运行游戏.bat`  （每次都会自动编译并运行）

---

## 🧩 运行方式（手动）

### 1) 编译
```bat
javac -encoding UTF-8 Snake.java
```

### 2) 运行
```bat
java -Dfile.encoding=UTF-8 Snake
```

> ⚠️ 请保证命令行编码为 UTF-8，否则中文可能显示异常。

---

## 🎮 控制方式

| 按键 | 功能 |
|------|------|
| **W** | 向上 | 
| **S** | 向下 |
| **A** | 向左 |
| **D** | 向右 |
| **空格** | 切换手动/自动模式 |
| **R** | 游戏结束后重新开始 |

---

## 🧠 模式说明

### 手动模式（默认）
- 使用 **WASD** 控制蛇移动。

### 自动模式
- 自动寻找最近食物的最短路径并自动移动。
- 使用 **BFS（广度优先搜索）** 实时计算路径。
- 会避开自身身体，直到吃到食物或无路可走。

> 💡 按 **空格** 切换模式，适合观察算法表现。

---

## 🕹️ 游戏规则

- 吃到食物：蛇变长，得分 +10
- 碰到自己：游戏结束
- 边界环绕：蛇出边界会从另一边出现，不会死亡

---

## 🎯 分数显示

游戏界面顶部会显示当前得分和模式状态。

---

## 🧱 自定义参数（可在代码中修改）

修改 `Snake.java` 中的常量即可调整游戏体验：

```java
GRID_SIZE = 20        // 网格单元大小（像素）
GRID_WIDTH = 30       // 游戏区域宽度（网格数）
GRID_HEIGHT = 20      // 游戏区域高度（网格数）
GAME_SPEED = 100      // 游戏更新间隔（毫秒）
```
